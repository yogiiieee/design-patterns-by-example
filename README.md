### What are Design Patterns?

Design patterns are reusable, proven solutions to common software design problems that help make code more flexible, maintainable, and understandable.

### Why are Design Patterns used?

Design patterns are used because they provide proven solutions that improve code quality, reusability, scalability, maintainability, and make systems easier to understand and extend.

### Categories of Design Patterns

Design patterns are broadly categorized as:

1. **Creational Patterns** - How objects are created
2. **Structural Patterns** - How objects are composed
3. **Behavioral Patterns** - How objects interact
